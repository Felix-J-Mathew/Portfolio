Featured Project

TruthLens

TruthLens is an analysis tool that can be used to check if an image is real or manipulated either through AI tools or image editing software.
The tool can also be used to fact check articles or news Via the help of reputable News APIs.
The tool uses Error Level Analysis(ELA), Metadata Analysis , Clone detection etc for image analysis.
The tool is build on python and uses libraries like NumPy, Pillow , OpenCV , fastapi etc.
